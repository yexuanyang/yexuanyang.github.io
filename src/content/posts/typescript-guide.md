# TypeScript 入门指南

TypeScript 作为 JavaScript 的超集，为 JS 添加了静态类型检查，极大提高了代码的可维护性和开发效率。

## 为什么要学习 TypeScript？

1. 更好的代码提示
2. 更少的运行时错误
3. 更容易重构
4. 更好的团队协作
